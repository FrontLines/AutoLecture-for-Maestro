The AutoLecture Voice Server is now running silently in your system tray and is configured to start automatically when you turn on your PC.

Final Step: Load the Chrome Extension

Because this is a custom-built extension, you just need to link it to Chrome once:

  1. Open Google Chrome and type chrome://extensions in the address bar.
  2. In the top right corner, turn on the Developer mode toggle.
  3. In the top left corner, click the Load unpacked button.
  4. A file browser will open. Paste this exact path into the address bar and press Enter:

    %LocalAppData%\AutoLecture\ChromeExtension

    (Or manually navigate to your AppData/Local/AutoLecture/ChromeExtension folder)
  5. Click Select Folder.

That's it! You can now close this window and pin the AutoLecture icon to your Chrome toolbar!
